First Home Buyer Benefits

What applies — and what to ask the agent

Use this at inspections. Which benefits apply depends on property type, price, and whether it has ever been occupied. Ask these questions on the spot.

FHBG — always use 5% deposit, no LMI Federal scheme. Cap $950k for Melbourne & Geelong. No income limit (removed Oct 2025). Formally apply through Luke before auction.
FHOG — new homes only $10,000 grant Victoria only. Price ≤$750k. Property must never have been occupied or sold as a residence. Not applicable to established homes above $750k.
Off-the-plan concession Stamp duty on land only Strata only (unit/apartment/townhouse with common property). Contract signed BEFORE completion. Extended to Oct 2026. Saves ~$25–45k.
FHB stamp duty exemption $0 or reduced duty Full exemption ≤$600k. Partial concession $600k–$750k. Nothing above $750k. Not relevant at our budget.
H&L packages $0 stamp duty Duty on land only. FHB on land ≤$600k = $0. Build costs never dutiable. Deprioritised: 12–18 months rent during construction absorbs saving.

Questions to ask the agent

  • "Has this property ever been occupied or sold as a residence?" If no → new home → FHOG ($10k) may apply if price ≤$750k.
  • "Is this a direct sale from the developer, or a private resale?" Developer sale of completed new build = new home for FHOG. Full stamp duty still applies if already built.
  • "Was this sold off-the-plan, or already completed when listed?" Off-the-plan + strata = stamp duty concession. Already built = no concession.
  • "Is there a body corporate / owners corporation?" Yes = strata = off-the-plan concession potentially eligible.
  • "What is the price guide / expected range?" Above $950k = FHBG cap breached. Under $750k new home = check FHOG.

Quick eligibility logic

  • Any property ≤$950k (Melb/Geelong): FHBG applies. 5% deposit, zero LMI. Always use.
  • New home, never occupied, price ≤$750k: Also get FHOG ($10k). Ask "has anyone ever lived here?"
  • Strata + signed before completion: Off-the-plan concession → duty on land value only. Saves ~$25–45k.
  • Completed developer sale (e.g. 23 Arbour Grove): Full stamp duty. FHOG may apply if ≤$750k and never occupied. Off-the-plan concession does NOT apply.
  • Established home above $750k (standard scenario): FHBG only. No other concessions.

Schemes verified May 2026. FHBG cap $950k for Melbourne and Geelong from 1 Oct 2025. Off-the-plan concession extended to 20 Oct 2026. Confirm specifics with Luke or conveyancer before signing.
